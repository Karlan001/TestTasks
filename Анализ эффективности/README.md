##Консольное приложение для анализа эффективности разработчиков
- Приложение сортирует данные от большего к меньшему по эффективности разработчиков 

###Для запуска приложения необходимо:
1. Установить зависимости из файла requirements.txt
   - Для установки зависимостей необходимо выполнить команду pip install requirements.txt
2. Открыть консоль
   - Ввести команду python3 main.py --files [имя_файла] --report [название отчета]

**Для просмотра возможных опций необходимо ввести команду _python3 main.py --help_**

###Для запуска тестов необходимо:
- Ввести команду pytest test.py
